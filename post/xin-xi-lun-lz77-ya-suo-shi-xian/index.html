LZ-77算法描述与实现流程

算法描述

LZ-77是由以色列人Abraham Lempel发表于1977年的一种无损压缩算法。LZ-77是典型的基于字典的压缩算法，如我们常用的ZIP数据压缩软件。本文会对LZ-77算法解释其原理与代码，主要以Python语言实现。

LZ-77的原理是将重复的语句替换成长度相对更短的三元组，如下

<d,l,c><d,l,c> <d,l,c>

其中ddd代表字符出现的距离，lll表示重复出现的长度，ccc代表重复出现的字符。比如对如下例子：

ABCABABCAB ABCAB

由于字符串ABABAB重复出现，就可以把第二次出现的ABABAB简记为

<3,2><3,2> <3,2>

而CCC第一次出现，可以将CCC记为

<0,0,C><0,0,C> <0,0,C>

所以上面的字符串经过压缩后得到

{<0,0,A>,<0,0,B>,<0,0,C>,<3,2>}\{<0,0,A>, <0,0,B>, <0,0,C>, <3,2> \} {<0,0,A>,<0,0,B>,<0,0,C>,<3,2>}

当字符串中重复的位数较多时，起到较好的压缩效果。而相应的解压缩过程是将上述过程逆着进行，一直压缩后的结果，逐步在字符串上添加字符，直到把所有的字符添加完成，具体逐步操作如下：

  1. A
  2. AB
  3. ABC
  4. ABCAB

在第四步中，找到3.中与新添加字符距离为3的字符，为A；长度为2，得到AB。将AB添加到3.结果的后面就得到了4.的结果。

算法实现

class LZ77:
    def __init__ ( self, inputStr ):
        self.inputStr = inputStr # 设置输入源
        self.lookaheadBufferSize = 5 # 设定查找缓冲区长度
        self.searchBufferSize = 3 # 设定先行缓冲区长度
        self.mainIndex = 0
        self.output = [] # 输出是一个列表

    def Match ( self ):
        # 初始化返回值
        position, length = 0, 0
        char = self.inputStr[ self.mainIndex ]
        # 这里i表示查找缓冲区的索引，倒着遍历
        for i in range ( self.mainIndex - 1, self.mainIndex - self.lookaheadBufferSize - 1, -1 ):
            # 边界判定
            if ( i < 0 ): break
            # l为本轮匹配到的长度，初始化为0
            l = 0
            # 匹配
            while ( self.inputStr[ i + l ] == self.inputStr[ self.mainIndex + l ] ):
                l += 1
                # 结束条件的判定
                # 1. 先行缓冲区的索引不能超过数组边界
                # 2. 查找缓冲区、先行缓冲区的索引不能超过本区的边界
                if ( self.mainIndex + l >= len( self.inputStr ) ): break
                if ( i + l >= self.mainIndex or l >= self.searchBufferSize ): break
            # 将匹配结果与保存的结果比较，决定是否更新
            if ( l >= length and l >= 1 ):
                length = l
                position = self.mainIndex - i
        # 返回匹配结果
        if ( position == 0 ): return ( position, length, char )
        else: return ( position, length )

    def encoding ( self ):
        # 编码过程。将所有输入字符串的元素按照规则进行匹配，并将结果保存在output数组中返回
        while ( self.mainIndex < len( self.inputStr ) ):
            rt = self.Match()
            self.mainIndex += rt[1]
            if ( rt[1] == 0 ): self.mainIndex += 1
            self.output.append( rt )
        return self.output

    def decoding ( self ):
        rt = ""
        for i in self.output:
            if ( i[0] == 0 ):
                rt += i[2]
            else:
                rt += rt[ len(rt) - i[0] : len(rt) - i[0] + i[1] ]
        return rt

用上面的代码对字符串"AABCBBABC"进行压缩，得到[(0, 0, 'A'), (1, 1), (0, 0, 'B'), (0, 0, 'C'), (2, 1), (3, 1), (5, 3)]

test = LZ77( "AABCBBABC" )
print( test.encoding() )
print( test.decoding() )
