有序对

由两个元素xxx与yyy按照一定顺序排列成的二元组称为一个有序对

< ⁣x,y ⁣><\!x,y\!> <x,y>

有序对 < ⁣x,y ⁣><\!x,y\!><x,y> 具有以下性质

x≠y⇒< ⁣x,y ⁣>  ≠  < ⁣y,x ⁣>x \neq y \quad \Rightarrow \quad <\!x,y\!> \; \neq \; <\!y,x\!> x​=y⇒<x,y>​=<y,x>

< ⁣x,y ⁣>  =  < ⁣u,v ⁣>⇔x=y ∧ u=v<\!x,y\!> \; = \; <\!u,v\!> \quad \Leftrightarrow \quad x=y \ \wedge \ u=v <x,y>=<u,v>⇔x=y ∧ u=v

笛卡儿积

对于集合AAA与集合BBB，由AAA中的元素作为第一元素，BBB总的元素为第二元素构成的有序对集合称为AAA与BBB的笛卡尔积，记作A×BA\times BA×B

A×B={< ⁣x,y ⁣>∣ x∈A ∧ y∈B}A \times B = \{ <\!x,y\!> \vert \ x \in A \ \wedge \ y \in B \} A×B={<x,y>∣ x∈A ∧ y∈B}

笛卡尔积具有以下性质

  • A×∅=∅A \times \varnothing = \varnothing A×∅=∅

  • A×(B ∪ C)=(A×B) ∪ (A×C)A \times (B\ \cup\ C) = (A \times B)\ \cup \ (A \times C) A×(B ∪ C)=(A×B) ∪ (A×C)

    (B ∪ C)×A=(B×A) ∪ (C×A)(B\ \cup\ C) \times A = (B \times A)\ \cup \ (C \times A) (B ∪ C)×A=(B×A) ∪ (C×A)

    A×(B ∩ C)=(A×B) ∩ (A×C)A \times (B\ \cap\ C) = (A \times B)\ \cap \ (A \times C) A×(B ∩ C)=(A×B) ∩ (A×C)

    (B ∩ C)×A=(B×A) ∩ (C×A)(B\ \cap\ C) \times A = (B \times A)\ \cap \ (C \times A) (B ∩ C)×A=(B×A) ∩ (C×A)

  • A⊆C ∧ B⊆D⇒A×B⊆C×DA \subseteq C \ \wedge \ B \subseteq D \quad \Rightarrow \quad A \times B \subseteq C \times D A⊆C ∧ B⊆D⇒A×B⊆C×D

二元关系

一个集合是二元关系，当它满足下列条件之一

  • 集合是空集(称为空关系)
  • 集合中的元素都是有序对

对于某个二元关系RRR，xRyxRyxRy表示

< ⁣x,y ⁣>  ∈R<\!x,y\!> \; \in R <x,y>∈R

对某个集合AAA，定义如下关系

  • 全域关系

    EA=A×A={< ⁣x,y ⁣>∣ x∈A ∧ y∈A}E_A = A \times A = \{ <\!x,y\!> \vert \ x \in A \ \wedge \ y \in A \} EA​=A×A={<x,y>∣ x∈A ∧ y∈A}

  • 恒等关系

    IA={< ⁣x,x ⁣>∣ x∈A}I_A = \{ <\!x,x\!> \vert \ x \in A \} IA​={<x,x>∣ x∈A}

  • 小于等于关系

    LA={< ⁣x,y ⁣>∣ x,y∈A ∧x⩽y}L_A = \{ <\!x,y\!> \vert \ x,y \in A \ \wedge x \leqslant y \} LA​={<x,y>∣ x,y∈A ∧x⩽y}

  • 整除关系

    DA={< ⁣x,y ⁣>∣ x,y∈A ∧x ∣ y}D_A = \{ <\!x,y\!> \vert \ x,y \in A \ \wedge x\, \vert \, y \} DA​={<x,y>∣ x,y∈A ∧x∣y}

  • 包含关系

    R⊆={< ⁣x,y ⁣>∣ x,y∈A ∧x⊆y}R_\subseteq = \{ <\!x,y\!> \vert \ x,y \in A \ \wedge x \subseteq y \} R⊆​={<x,y>∣ x,y∈A ∧x⊆y}

关系的性质

关系的性质主要分为以下五种

  • 自反性

    ∀x ( x∈A →  < ⁣x,x ⁣> ∈R )\forall x \ (\ x\in A\ \rightarrow\; <\!x,x\!> \, \in R\ ) ∀x ( x∈A →<x,x>∈R )

  • 反自反性

    ∀x ( x∈A →  < ⁣x,x ⁣> ∉R )\forall x \ (\ x\in A\ \rightarrow\; <\!x,x\!> \, \notin R\ ) ∀x ( x∈A →<x,x>∈/​R )

  • 对称性

    ∀x ∀y ( x,y∈A ∧< ⁣x,y ⁣> ∈R →  < ⁣y,x ⁣> ∈R )\forall x \ \forall y \ (\ x,y\in A \ \wedge <\!x,y\!> \, \in R\ \rightarrow\; <\!y,x\!> \, \in R\ ) ∀x ∀y ( x,y∈A ∧<x,y>∈R →<y,x>∈R )

  • 反对称性

    ∀x ∀y ( x,y∈A  ∧< ⁣x,y ⁣> ∈R  ∧< ⁣y,x ⁣> ∈R→  x=y )\forall x \ \forall y \ (\ x,y\in A \; \wedge <\!x,y\!> \, \in R\; \wedge <\!y,x\!> \, \in R \rightarrow\;x=y \ ) ∀x ∀y ( x,y∈A∧<x,y>∈R∧<y,x>∈R→x=y )

  • 传递性

    ∀x ∀y ∀z ( x,y,z∈A ∧< ⁣x,y ⁣> ∈R < ⁣y,z ⁣> ∈R →  < ⁣x,z ⁣> ∈R )\forall x \ \forall y \ \forall z \ (\ x,y,z\in A \ \wedge <\!x,y\!> \, \in R\ <\!y,z\!> \, \in R\ \rightarrow\; <\!x,z\!> \, \in R\ ) ∀x ∀y ∀z ( x,y,z∈A ∧<x,y>∈R <y,z>∈R →<x,z>∈R )

等价关系

对于在集合AAA上的关系RRR，如果RRR是自反的、对称的和传递的，则称RRR为AAA的等价关系

< ⁣x,y ⁣>  ∈R⇒x∼y<\!x,y\!>\; \in R \quad \Rightarrow \quad x \sim y <x,y>∈R⇒x∼y

若非空集合AAA具有等价关系RRR,则AAA中的元素，对于∀x∈A\forall x\in A∀x∈A，都构成一个xxx关于RRR的等价类

在上述条件下，以RRR的所有等价类作为元素的集合称为AAA的关于RRR的商集

  • ∅∉π\varnothing \notin \pi ∅∈/​π
